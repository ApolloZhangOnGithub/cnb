# #167 Demo: 独立研究 — 开放问题的实验驱动探索

**State:** OPEN
**Labels:** enhancement, phase:2, experiment

---

## 目标

给一个开放研究问题，cnb 团队自主设计实验、收集数据、分析结果、产出研究报告。

## 候选研究问题

- "RAG vs fine-tuning 在小数据集上的表现对比"
- "不同 prompt 策略对代码生成质量的影响"
- "multi-agent 协调开销与团队规模的关系"（meta — 用 cnb 研究 cnb）

## 为什么这是好 demo

- 展示 cnb 处理非确定性任务的能力
- 假设拆分、实验设计、数据收集、分析、写作 — 每步不同 owner
- 研究过程本身就是 ownership 和交接的最好案例
- meta 研究（用 cnb 研究 multi-agent）特别有说服力

## 展示重点

- experiment 标签的 issue 管理
- 假设 → 实验 → 结果 → 结论的完整流程
- 跨 session 的研究笔记积累（daily report）
- 最终产出：可发布质量的研究报告
