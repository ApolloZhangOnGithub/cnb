# #56 Maintain GitHub Wiki: sync project knowledge from scattered docs

**State:** OPEN
**Labels:** phase:2, infra

---

## Summary

项目知识散落在 ROADMAP.md、CLAUDE.md、CHANGELOG.md、bulletin.md、日报、issue 等多处。GitHub Wiki 应该作为统一的、面向外部的知识入口。

## ROADMAP 关系

与 #54（context health）相关。Wiki 是文档整理的一部分。

## 需要维护的内容

- 架构概览（从 CLAUDE.md 和 README 提炼）
- 命令速查（从 board help 生成）
- Ownership map（从 board own map 生成）
- 实验记录（#50-#55 的实验结果）
- 版本变更摘要（从 CHANGELOG 提炼）

## 注意

Wiki 维护是持续性工作，不是一次性任务。需要指定 owner 或纳入定期 audit 流程。
