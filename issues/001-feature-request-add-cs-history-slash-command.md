# #1 Feature request: add /cs-history slash command

**State:** CLOSED

---

## Request

Add a `/cs-history` skill/slash command so the lead can view the full message history directly, without needing to manually run `board --as lead history`.

Currently available `/cs-*` commands don't cover this:
- `/cs-team` only shows recent messages
- `/cs-inbox` only shows unread messages to lead

## Suggested behavior

`/cs-history` runs `board --as lead history` and displays the full conversation log to the user.
