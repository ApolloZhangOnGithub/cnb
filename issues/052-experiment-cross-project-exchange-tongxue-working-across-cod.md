# #52 Experiment: cross-project exchange — tongxue working across codebases

**State:** OPEN
**Labels:** experiment, org-design

---

## Hypothesis

同学在不同项目间交流（团建）或临时借调（跳槽）可能带来新视角、发现共性问题、促进知识传播。

## ROADMAP 关系

与 #47（跨项目统一身份）相关。需要统一身份才能让同学在项目间流动。

## 实验设计

1. 让一个 cnb 同学去 TokenDance_BBS 或 quest_to_avp 做一个 shift
2. 观察：是否发现了原项目同学没注意到的问题？是否把一个项目的好实践带到了另一个？
3. 反向：让外部项目的"新人视角"审视 cnb 代码

## 验证标准

- 跨项目同学是否产出了有价值的 issue 或 PR
- 上下文切换成本有多高（通过日报和 task 完成速度衡量）

## 前置条件

- #47 跨项目统一身份（或至少手动配置临时身份）
