# #163 CLI 输出美化：添加颜色和视觉层次

**State:** OPEN
**Labels:** enhancement, phase:1, infra

---

## 问题

Python 端所有 board 命令输出全是裸 `print()`，没有颜色和视觉层次。`bin/cnb` bash 层有基础 ANSI 色彩变量（`$B $G $Y $N`），但 Python 模块完全没有对应设施。

当前输出：
- 标题（`=== Board ===`）和正文视觉上无差异
- `OK sent` / `ERROR:` 没有颜色区分
- session 状态（`● working` / `○ shell` / `· offline`）全是同色
- active/pending/done 任务无颜色区分
- 只有 `board_pending.py` 有 emoji，其余全靠空格对齐

## 方案

1. 新建 `lib/fmt.py`，提供 ANSI 颜色常量和工具函数（TTY 检测自动降级）
2. 逐模块应用到 `board_view.py`, `board_display.py`, `board_msg.py`, `board_task.py`, `board_pending.py`

## 不做

- 不引入外部依赖（不用 `rich` / `colorama`）
- 不改变输出的信息内容，只加视觉装饰
- 不影响管道/重定向场景（非 TTY 自动无色）

## ROADMAP 关系

与现有 roadmap 项目无冲突。属于运营基础的 DX 改善。
