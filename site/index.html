Install

One global command, then run the team from your project.

npm install -g claude-nb

The npm package is claude-nb; it installs the cnb command. Do not install cnb from npm, which is an unrelated package.

01

Activate governance

Run /cnb inside Claude Code to bring the device supervisor online.

02

Launch the board

Run cnb in a project to initialize tmux sessions, storage, and routing.

03

Keep work accountable

Assign scopes, route tasks to owners, and preserve handoff evidence between runs.

Why it exists

Session managers run agents. cnb manages the organization around them.

Persistent ownership

Module responsibility survives restarts through ownership maps, session notes, daily reports, and routing.

Board-based coordination

Messages, tasks, status, pending actions, and inboxes live in local SQLite instead of transcript sprawl.

Local operational control

tmux sessions, local files, and explicit checks keep the runtime inspectable on a trusted workstation.

Feishu as the front door

Wake the Mac supervisor from Feishu, preserve quoted-message context, and notify mobile only on final results.

Verification-first handoff

Owners report what changed, which checks ran, and what remains blocked before the next session takes over.

Flow

The durable path is outside the context window.

01

Supervisor

User-facing session receives priorities and high-risk confirmations.

02

Board

03

Tongxue

Claude Code or Codex sessions work independently through scoped ownership.

04

Handoff

Reports, checks, issues, and PR evidence preserve continuity for the next run.

Feishu integration

Your AI team can be reached from the chat app people already use.

CNB routes Feishu messages into the local device supervisor on the Mac, keeps reply and quote context, and treats mobile notifications as a scarce channel: routine work stays quiet until the final result.

01 Natural-language dispatch

Users talk in Feishu; the supervisor chooses the local cnb, board, and swarm commands.

02 Quiet by default

iOS is notified on the final reply, while optional live mode updates one readable activity card.

03 Live TUI when needed

A tokenized Web TUI refreshes at 250 ms for inspection without turning Feishu into a video stream.
